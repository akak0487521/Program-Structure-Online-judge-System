# Program-Structure-Online-judge-System
Overview  A rule-based automatic grading system for evaluating code structure and UML class design in object-oriented programming education.

# My Contributions

- Developed rule-based evaluation engine
- Implemented UML class diagram similarity matching
- Integrated RESTful API and grading workflow

- Applied in university-level programming courses
